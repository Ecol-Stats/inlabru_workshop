\section{Welcome to the course!}\label{welcome-to-the-course}

\begin{itemize}
\tightlist
\item
  Welcome to the \texttt{inlabru} workshop!
\item
  The goal of this workshop is to introduce new users to the
  \texttt{inlabru} software for Bayesian spatial modelling using the
  integrated nested Laplace approximation (INLA).
\item
  The workshop is intended for applied statisticians and quantitative
  ecologists who are interested in efficiently modelling spatial data-
  No prior knowledge of \texttt{R-INLA} is required.
\end{itemize}

\section{Learning Objectives for the
workshop}\label{learning-objectives-for-the-workshop}

At the end of the workshop, participants will be able to:

\begin{itemize}
\tightlist
\item
  Understand the \texttt{inlabru} workflow and software principles.
\item
  Fit commonly used models such as Linear models, LMM, GLM and GAMs
  using \texttt{inlabru}.
\item
  Describe different types of spatial data and fit appropriate spatial
  models.
\item
  Interpret model outputs and create spatial maps and predictions.
\item
  Implement a complete, reproducible spatial analysis workflow from data
  preparation to result visualization.
\end{itemize}
